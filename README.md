<h2>💻FLOW</h2>
<h4>
  1. 로그인 (C:CHOOSE 사용/ 로그인 예외처리)<br><br>
  2. 입장화면에서 입장<br><br>
  3. 게시판 등록하기(등록시 작성일, 닉네임 입력하지 않아도 됨. Readonly처리함)<br><br>
  4. 게시판 수정하기(등록한 글을 가져와서 수정할 수 있도록 함.)<br><br>
  5. 게시판 삭제하기(작성한 게시글을 삭제함)<br><br>
</h4>
